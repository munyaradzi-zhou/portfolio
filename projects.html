FarmSmart

Real-Time NL Analytics over IoT Agricultural Streams

A complete natural language analytics system over real-time IoT sensor streams. Farmers ask questions like "What was the average soil moisture in the north field last week?" and receive instant answers derived from live sensor data processed through Apache Spark Structured Streaming.

The system features a custom NL-to-query translator with intent classification (TF-IDF + Logistic Regression), slot extraction, and query generation that routes through real data repositories — InfluxDB for time-series sensor data, PostgreSQL for field metadata, and a real-time alert engine for anomaly detection.

Architecture Highlights

  • Apache Kafka for sensor data ingestion from IoT devices
  • Spark Structured Streaming with sliding window aggregations
  • Intent classification + slot extraction NLP pipeline
  • InfluxDB time-series storage with PostgreSQL field metadata
  • FastAPI backend with React dashboard frontend
  • Kubernetes-ready with Docker Compose orchestration

AgriSage

Graph-Native RAG for Agricultural Knowledge Retrieval

A graph-native Retrieval-Augmented Generation system that builds agricultural knowledge graphs from crop manuals, spraying guidelines, and pest management documents. Uses the same ReAct decomposition and schema-aware Cypher generation architecture from my master's thesis, adapted for the agricultural domain.

The system ingests unstructured agricultural documents, extracts entities (crops, pests, diseases, treatments, parameters), builds a structured Neo4j knowledge graph, and enables multi-hop reasoning through conversational queries. A hybrid retrieval approach combines graph-native traversal with vector similarity for comprehensive coverage.

Architecture Highlights

  • Neo4j knowledge graph with agricultural ontology (Crop, Pest, Disease, Treatment, Parameter nodes)
  • ReAct-based query decomposition for complex multi-hop questions
  • Schema-aware Cypher generation with few-shot domain-specific prompting
  • Hybrid graph + vector (ChromaDB) retrieval pipeline
  • Document ingestion with entity extraction and relationship mapping
  • Streamlit conversational interface with graph visualization

CropCommand

Natural Language Task Planning for Agricultural Robots

A natural language task planning system that translates farming commands into structured, constraint-aware execution plans for agricultural robots. Farmers speak naturally — "Spray the northern tomato field before the rain, avoiding the organic section" — and receive validated, executable robot task plans with full constraint reasoning.

The system parses natural language commands, decomposes them into atomic tasks, applies domain-specific constraints (weather windows, crop growth stages, terrain limitations, robot capabilities), generates optimised schedules, and presents plans for human review with natural language explanations and iterative refinement.

Architecture Highlights

  • NL intent parsing with entity extraction and command resolution
  • Hierarchical task decomposition into atomic robot actions
  • Multi-dimensional constraint engine (weather, terrain, crop state, robot capabilities)
  • Plan generation with scheduling optimisation and confidence scoring
  • Farm environment and robot state simulation
  • Human-in-the-loop feedback with NL plan refinement and explanation generation

Football Agentic RAG System

Multi-Agent Temporal Knowledge Graphs for Sports Analytics

An AI-powered football analytics system that combines temporal knowledge graphs, multi-agent orchestration, and RAG for intelligent question answering over 32 seasons (1993-2025) of European football data. Users ask natural language questions like "Who was Manchester United's top scorer in 2023?" and receive accurate answers with visualizations, sourced from knowledge graphs, structured databases, and vector stores.

The system features an orchestrator agent that routes queries to specialized sub-agents: temporal normalization for date parsing, knowledge graph queries for relationship traversal, stats aggregation for numerical analysis, explanation RAG for contextual insights, and visualization generation for chart specs. Includes comprehensive provenance tracking, caching, and multi-turn conversation support.

Architecture Highlights

  • Multi-agent orchestration with 6 specialized agents (orchestrator, temporal normalizer, router, KG query, stats, RAG, visualization)
  • Temporal knowledge graph (Neo4j) with 32 seasons of football data
  • Hybrid retrieval: graph traversal + vector search (Chroma) + SQL aggregation (PostgreSQL)
  • Multi-turn conversation with context tracking (teams, players, competitions, seasons)
  • Provenance tracking and anti-hallucination verification for reliable answers
  • FastAPI backend with Streamlit conversational interface

Technical Contributions

Demonstrates advanced agentic AI patterns: query routing, temporal reasoning, hybrid retrieval, session management, and human-in-the-loop conversation — applicable to conversational systems across domains (agriculture, healthcare, finance) requiring multi-source data integration.

ChurnSense

Conversational AI for Customer Intelligence & Retention

A production-grade conversational AI system for customer churn prediction, analytics, and retention strategy. Business users ask questions like "Show me high-risk customers", "Why is customer 10025 at risk?", or "Give me retention strategies for premium customers" and receive intelligent answers with automatic visualizations and explainable insights.

Features a multi-agent orchestration architecture with 6 specialized AI agents: Orchestrator for coordination, Query Router for intent classification, ML Prediction Agent (99.9% accuracy), Churn Analytics Agent for segment analysis, Explainability Agent with SHAP-based feature importance and natural language generation, and Visualization Agent for auto chart type detection.

Architecture Highlights

  • Multi-agent orchestration with 6 specialized AI agents
  • LLM-powered intent classification and query enrichment (OpenAI)
  • Multi-turn dialogue with context tracking and pronoun resolution
  • SHAP-based explainability + natural language generation
  • Random Forest ML model with 99.9% prediction accuracy
  • Auto-detecting visualization agent (6 chart types)
  • Streamlit conversational interface + legacy Flask API
  • Redis session management for distributed conversations
  • Docker containerization with Prometheus monitoring
